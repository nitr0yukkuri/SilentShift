# SilentShift（サイレント・シフト）

## 🚀 コンセプト
SilentShiftは、デジタル・コミュニケーション空間における音響的空白、すなわち**「沈黙という負の圧力」をGo言語の超並行処理エンジンでリアルタイム捕捉し、Gemini APIによる対話履歴の文脈解析から導き出された「最適化された不条理（クソゲー）」を即時介入させることで、ソーシャル・エントロピーの停滞を強制的に破壊する認知転換型コミュニケーション・リアクター**である。

「気まずさ」という主観的な心理的摩擦を、非言語的な同期作業（クソゲー）という共通の物理的課題へと置換・転嫁し、参加者の意識を「対峙」から**「共闘」へとリダイレクトする。これにより、沈黙による関係性の硬直を、笑いを介した情緒的融和へと昇華させる、チームビルディングのための強制介入型ソーシャル・アクセラレータ**である。

## 📋 要件定義

### ターゲットと解決する課題
* **課題**: Discord等のボイスチャットにおいて発生する「気まずい沈黙（ソーシャル・エントロピーの停滞）」によるコミュニケーションの硬直。
* **解決策**: 沈黙をリアルタイムに検知し、直前の会話文脈に合致した（しかし絶妙に不条理な）マルチプレイ3Dゲームへとユーザーを強制誘導。意識を「対峙」から「共闘・笑い」へと強制リダイレクトする。

### 機能要件
1. **ボイスチャンネル監視機能**: DiscordのRTPパケットを常時監視し、設定された秒数（例: 7秒）の無音状態を検知する。
2. **コンテキスト・キャッシング**: 直近のテキストチャット（または音声認識ログ）をインメモリで一時保存する。
3. **AI文脈解析エンジン**: 無音検知時、キャッシュされた会話ログをGemini APIに送信し、文脈から「気まずさスコア」の算出と「最適化されたクソゲーテーマ」の動的選定を行う。
4. **強制介入機能**: Discordのテキストチャンネルに、AIが生成した煽りメッセージと、一意に発行されたゲームルーム（Webアプリ）のURLを自動送信する。
5. **マルチプレイ3D空間**: 発行されたURLからアクセスしたユーザー同士で、WebGLによる3Dミニゲーム（物理演算ベース）をリアルタイムに同期・プレイできる。

### 非機能要件
1. **リアルタイム性**: 音声パケット監視時のI/O遅延をゼロにし、沈黙検知からDiscordへのURL投下までをシームレスに行う（Gemini 3.1 Flashによる推論の高速化）。
2. **スケーラビリティ**: 複数サーバー・複数VCでの同時稼働に耐えうる、Goルーチンの安全なライフサイクル管理（メモリリークの防止）。
3. **可用性**: WebSocket通信を維持するため、Botサーバーはゼロスケールさせず常時稼働（CPU Always allocated）とする。

## 🛠 技術スタック

### バックエンド（Bot & Core Logic）
* **言語**: Go
* **ライブラリ**: `bwmarrin/discordgo`
* **特徴**: Goroutineを用いた非同期パケット監視、Ring Bufferと `sync.RWMutex` によるスレッドセーフなインメモリログ管理。

### AI・文脈解析
* **API**: Gemini 3.1 Flash API
* **特徴**: 構造化プロンプト（Chain of Thought誘導型JSON出力）による、高速かつ精度の高い「大喜利・テーマ生成」ロジック。

### フロントエンド（3D Kusoge Dimension）
* **フレームワーク**: Next.js (App Router)
* **描画エンジン**: Three.js / React Three Fiber (R3F)
* **状態同期**: Socket.io / Zustand
* **特徴**: URLパラメータによるルーム生成、ブラウザ上でのミリ秒単位の物理座標同期。

## 🏗 システムフロー
1. **[Go]** Discord VCのRTPパケットを常時監視。
2. **[Go]** N秒間の無音（沈黙）を検知。
3. **[Go → Gemini]** 直近の会話ログをGemini 3.1 Flashに送信し、文脈解析。
4. **[Gemini → Go]** 文脈に合ったクソゲーのテーマとパラメータをJSONで返却。
5. **[Go → Discord]** チャンネルに介入メッセージとWebアプリ（ゲームルーム）のURLを送信。
6. **[Next.js + R3F]** ユーザーがURLを開き、WebGLベースの3D空間でリアルタイムマルチプレイを開始。
